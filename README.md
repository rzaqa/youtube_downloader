                         # 🎥 YouTube Video Downloader (GUI)

A simple desktop application for downloading YouTube videos using a graphical interface built with `tkinter` and `yt-dlp`.

---

## 🚀 Features

- 🖱️ User-friendly interface (built with Tkinter)
- 📥 Download videos in the best available MP4 format
- 📊 Real-time download progress displayed in window
- 🔗 Supports single YouTube video URLs
- ✅ Based on the reliable [`yt-dlp`](https://github.com/yt-dlp/yt-dlp) backend
- 🎨 Clean layout with YouTube-style branding

---

## 📦 Requirements

- Python 3.8+
- [yt-dlp](https://github.com/yt-dlp/yt-dlp)
- [Pillow](https://pypi.org/project/Pillow/) (for image support)

### ✅ Install dependencies:

```bash
pip install yt-dlp pillow
```

### ✅  How to Run
```bash
git clone https://github.com/your-username/youtube-downloader-gui.git
```

```bash
cd youtube-downloader-gui
```

### ✅  Run the application:
```bash
python main.py
```
